# public/images — генерируемая папка

Здесь лежит ВСЁ: и фото/svg, обработанные из src/images/ (плагин plugins/images.js),
и собранный иконный спрайт sprite.svg (плагин plugins/svg-sprite.js).

Не редактируй и не клади сюда файлы руками — они перезапишутся при следующем
сохранении файла в src/images/ или src/icons/, либо при запуске npm run dev / build / images.
